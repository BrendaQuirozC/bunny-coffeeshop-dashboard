
  # Data Visualization Dashboard

  This is a code bundle for a Data Visualization Dashboard of a coffe shop. It shows data analitics to help the bussiness with better decisions.

  It was designed using figma and developed with React.

  The dashboard includes interactive charts tracking coffee sales, customer demographics, hourly/daily patterns, seasonal trends, and ML-based predictions. It features KPI cards showing key metrics, multiple chart types (bar, pie, line, area), and three main sections accessible via tabs: Overview for quick insights, Analytics for detailed patterns, and ML Predictions for future forecasting with actionable recommendations.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
